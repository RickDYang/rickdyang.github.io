编译优化TensorFlow

发表于 2017-05-25 | 分类于 备忘 |

TensorFlow release版没有对特定硬件进行编译优化，如果要优化TensorFlow的运行效率，需要自己编译。
这里记录一下过程以备忘。

前言

如果你运行TensorFlow时，出现下面的Warning，那么表明TensorFlow没有针对你的CPU进行编译。恭喜你有机会可以尝试编译TensorFlow来优化。

Installing TensorFlow from Sources

按照TensorFlow官网上的Installing TensorFlow from Sources一步一步来，当然请仔细阅读第一段了解其风险。
另外有几点细节需要特别说明。
如果你之前安装过TensorFlow with GPU，那么就不需要特别的准备了。
注意配置前，需要安装Bazel，按官网Install Bazel on Ubuntu来
运行./configure进行配置的时候，注意检查一下依赖的路径是否正确。
走到下面这步时，需要到Nvidia官网上检查一下你的显卡对应的compute capability，我的GTX 1050是6.1
